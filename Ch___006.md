### tar

`tar -cf`  // 创建 tar 包

`tar -tf`	// 将包内容列出

`tar -tvf`	// 将包内容列出（包含详细消息）

`tar -rf`	// 向包内添加

`tar -rvf`	// 向包内添加

`tar -xf`		// 导出全部

`tar -xvf`	// 导出指定文件

`tar -Af`	// 拼接两个 tar 包

`tar -f xxxx.tar  --delete xxxx xx`  // 删除

`tar -acvf xxx.tar.gz  xxx`  // 打包并压缩成 gzip 格式
